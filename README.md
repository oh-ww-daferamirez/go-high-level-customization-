# ARM - GoHighLevel Custom Skin (v33.3 / v16.2)

Este repositorio contiene la personalización visual (Skin) para GoHighLevel.
**Versión 33.3 / JS 16.2 (Icon Neutral Mode)**: Mantiene visibilidad (opacidad) pero deja iconos en estilo nativo/neutro, evitando recolorados que rompen legibilidad.

## 🚀 Instalación en GoHighLevel

Para aplicar este diseño a tu agencia o sub-cuenta, ve a:
**Settings** > **Company** (o Location) > **Custom Code**.

### 1. Head Tracking Code (CSS v33.3)
```html
<link rel="stylesheet" href="https://go-high-level-customization.vercel.app/custom.css?v=33.3">
```

### 2. Body Tracking Code (JS v16.2)
```html
<script src="https://go-high-level-customization.vercel.app/custom.js?v=16.2"></script>
```
