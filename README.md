# ARM - GoHighLevel Custom Skin (v33.1 / v16.0)

Este repositorio contiene la personalización visual (Skin) para GoHighLevel.
**Versión 33.1 (Sidebar Contrast Recovery)**: Hotfix de contraste del sidebar para forzar visibilidad de texto/iconos y neutralizar opacidades heredadas de GHL.

## 🚀 Instalación en GoHighLevel

Para aplicar este diseño a tu agencia o sub-cuenta, ve a:
**Settings** > **Company** (o Location) > **Custom Code**.

### 1. Head Tracking Code (CSS v33.1)
```html
<link rel="stylesheet" href="https://go-high-level-customization.vercel.app/custom.css?v=33.1">
```

### 2. Body Tracking Code (JS v16.0)
*(Sin cambios desde v16.0)*
```html
<script src="https://go-high-level-customization.vercel.app/custom.js?v=16.0"></script>
```
