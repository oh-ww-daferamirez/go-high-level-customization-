# ARM - GoHighLevel Custom Skin (v33.2 / v16.1)

Este repositorio contiene la personalización visual (Skin) para GoHighLevel.
**Versión 33.2 / JS 16.1 (Visibility Recovery)**: Refuerzo combinado CSS + JS con reaplicación por `MutationObserver` para corregir opacidad/color en sidebar, header e iconos de cards tras re-renders de GHL.

## 🚀 Instalación en GoHighLevel

Para aplicar este diseño a tu agencia o sub-cuenta, ve a:
**Settings** > **Company** (o Location) > **Custom Code**.

### 1. Head Tracking Code (CSS v33.2)
```html
<link rel="stylesheet" href="https://go-high-level-customization.vercel.app/custom.css?v=33.2">
```

### 2. Body Tracking Code (JS v16.1)
```html
<script src="https://go-high-level-customization.vercel.app/custom.js?v=16.1"></script>
```
